\doxysection*{Types}

\mbox{\hyperlink{Types}{Types}}

Every form of data is a type. By extension, the same is true for functions. Built in types\+:
\begin{DoxyItemize}
\item null
\item boolean
\item int8
\item int64
\item float64
\item string
\item array
\item vector
\item set
\item tuple
\item dictionary
\end{DoxyItemize}

Types are based off of C++ types.

See \href{https://en.cppreference.com/w/cpp/language/types}{\texttt{ https\+://en.\+cppreference.\+com/w/cpp/language/types}}\hypertarget{Types_Integers}{}\doxysection{Integers}\label{Types_Integers}
All integers are signed. See \mbox{\hyperlink{Types_Unsigned}{Unsigned}} for more information.

There are two integer types\+:
\begin{DoxyItemize}
\item int8 is 8 bit. (-\/128 to +127)
\item int64 is 64 bit. (-\/9,223,372,036,854,775,808 to +9,223,372,036,854,775,807)
\end{DoxyItemize}

{\itshape Tip\+:} Only use int8 for space conserving situations, provided all values will always fall within limits. Else, use int64.
