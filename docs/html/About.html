About bill

Name:

Bill

  • Beginner
  • Intermediate
  • Learning
  • Language

Purpose:

It is a general purpose statically typed, easy to learn and use language.

Key Uses:

  • Learning basic level programming.
  • Pursuing enthusiast programming.

Features:

  • portable
  • a variety of useful data types
  • extend-able data types
  • simple to use
  • constants
  • ease of static typing
  • strong static typing

FAQ

  • Why is it called Bill?
     Bill is named after Bill Smith, who encouraged the language's creator to learn programming as a teenager.

  • Is Bill Object Oriented?
    As there are numerous definitions of "bill Object Oriented" accepted by the programming community, it is impossible to anser all of the definitions. (See next question.)

  • Can I define a class and create an instance of it?
    This is planned for a future version of Bill. At this time, it is not possible.

  • How can I build the documentation, on a Linux based system?
    It will be neccessary to install the following:
    • doxygen
    • graphviz

    • texlive-fonts-recommended
