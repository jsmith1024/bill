\doxyparagraph*{Purpose\+:}

It is a general purpose statically typed, with dynamic types, easy to learn and use language.

\doxyparagraph*{Key uses\+:}


\begin{DoxyItemize}
\item Learning basic level programming.
\item Pursuing enthusiast programming.
\end{DoxyItemize}

\doxyparagraph*{Features\+:}


\begin{DoxyItemize}
\item portable
\item a variety of useful data types
\item extend-\/able data types
\item simple to use
\item constants
\item ease of static typing
\item strong static typing
\end{DoxyItemize}



\mbox{\hyperlink{}{Hello World }}

~~~~~~~ \char`\"{}\+Hello World\char`\"{} is traditionally the first program one writes in a new language. That makes it a good starting point.


\begin{DoxyVerbInclude}
#!/usr/bin/env bill

# hello.bill
# aka hello world

fun main():no_value
{
    writeln("Hello World")
    exit                    // defaults to 0
}

\end{DoxyVerbInclude}
 hello.\+bill

~~~~~~~ Of course, the first few lines are unnecessary. However, declaring a main function is required. For details, on that, in shapter 6 (\mbox{\hyperlink{Functions}{Functions}}).

To compile and run\+:


\begin{DoxyVerbInclude}
prompt> bill hello.bill
\end{DoxyVerbInclude}


If you operating system supports shebangs, the following will work\+:


\begin{DoxyVerbInclude}
prompt> hello.bill
\end{DoxyVerbInclude}
