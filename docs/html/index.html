Introduction

Name:

Bill - Beginner - Intermediate - Learning - Language

Purpose:

It is a general purpose statically typed, easy to learn and use language.

Key uses:

- Learning basic level programming. - Pursuing enthusiast programming.

Features:

- portable - a variety of useful data types - extend-able data types - simple to use - constants - ease of static typing - strong static typing
